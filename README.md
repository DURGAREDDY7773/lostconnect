# LostConnect

LostConnect is a Java Android lost-and-found app using SQLite with Google Maps, Places autocomplete, current location lookup, map markers, and radius-based search.

## Features

- Add lost or found item posts with name, description, phone, image, address, latitude, longitude, date, and timestamp.
- Select addresses with Google Places Autocomplete.
- Fill the address from the device current location using Fused Location Provider and Geocoder.
- Show all saved items on a Google Map.
- Filter map markers by radius in kilometres from the user's current location.
- Red markers represent lost items and green markers represent found items.

## Setup

1. Open the project in Android Studio.
2. Add a Google Maps/Places API key in `app/src/main/res/values/strings.xml` by replacing `YOUR_API_KEY`.
3. Sync Gradle and run the app on a device or emulator with Google Play services.
